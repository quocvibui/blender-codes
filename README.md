## red_circle: python script
Moving a red circle around using the KORG nanoPAD2 midi controller, reacts to microphone.

- Installed python libraries: mido, p5, sounddevice
- For installing p5, git clone the project source code, go to requirements.txt and delete:
  PyOpenGL-accelerate==3.1.7. This package seems most likely as an optimization package for speed. Then do: <code>pip install p5 --no-dependencies</code>
https://github.com/p5py/p5/issues/439
